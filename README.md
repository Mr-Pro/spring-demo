## 简单的一些关于spring的小示例

主要包括如下部分：
* spring-base 
  - 通过ClassPathApplicationContext获取Bean
  - 通过xml简单装配Bean
  - AOP示例
  
* spring-tx
  - 主要通过xml配置Jdbc事务
  - 通过DruidDataSource连接数据库
  - 使用JdbcTemplate操作
  
* spring-annotation
  - 使用注解装配Bean
  - 使用注解完成AOP操作
  - 使用注解完成事务操作
  
* spring简单登录模块

  - 通过登录注册来对spring的基础进行应用
  - 使用注解+部分配置
  - 包含SpringMVC的简单实用
